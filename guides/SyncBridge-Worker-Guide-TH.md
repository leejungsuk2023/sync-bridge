# คู่มือการใช้งาน SyncBridge — สำหรับพนักงาน (Worker)

**SyncBridge** คือระบบซิงค์งานแบบเรียลไทม์ระหว่างบริษัทลูกค้า (โรงพยาบาลในเกาหลี) กับพนักงานทำงานทางไกล (ไทย)
คู่มือฉบับนี้สำหรับ **พนักงาน (Worker)** ที่ใช้ **Desktop App** หรือ **Chrome Extension** ในการรับ-ส่งงานและสื่อสารกับลูกค้า

---

## ส่วนที่ 1: Desktop App (แอปตั้งโต๊ะ)

### 1.1 การติดตั้ง

**macOS:**
1. ดาวน์โหลดไฟล์ `.dmg` จากลิงก์ที่ผู้ดูแลส่งให้
2. เปิดไฟล์ → ลากไอคอน SyncBridge ไปที่โฟลเดอร์ Applications
3. เปิดแอป SyncBridge จาก Applications

**Windows:**
1. ดาวน์โหลดไฟล์ `.exe` จากลิงก์ที่ผู้ดูแลส่งให้
2. ดับเบิลคลิกเพื่อติดตั้ง → ทำตามขั้นตอน
3. เปิดแอป SyncBridge จากเมนู Start หรือ Desktop shortcut

> แอปจะอัปเดตเวอร์ชันใหม่อัตโนมัติ เมื่อมีเวอร์ชันใหม่จะมีข้อความแจ้งเตือน

### 1.2 การเข้าสู่ระบบ

1. เปิดแอป SyncBridge
2. ใส่ **อีเมล** และ **รหัสผ่าน** ที่ได้รับจากผู้ดูแล
3. คลิก **เข้าสู่ระบบ**

> หากลืมรหัสผ่าน กรุณาติดต่อผู้ดูแลระบบ (BBG Admin)

---

### 1.3 การลงเวลาทำงาน

ที่ส่วนหัวของแอปจะมีปุ่มสถานะ 3 แบบ:

| ปุ่ม | ความหมาย | เมื่อไหร่ต้องกด |
|------|----------|----------------|
| **เข้างาน (출근)** 🟢 | กำลังทำงาน | กดเมื่อเริ่มทำงาน |
| **ไม่อยู่ (자리비움)** 🟡 | ลุกจากที่นั่งชั่วคราว | กดเมื่อต้องลุกไปทำอย่างอื่นสั้นๆ |
| **เลิกงาน (퇴근)** ⚪ | จบการทำงาน | กดเมื่อจบกะ/เลิกงาน |

- ใต้ปุ่มจะมี **ตัวนับเวลา** แสดงระยะเวลาที่อยู่สถานะนั้น
- เวลาจะถูกบันทึกในระบบทันที ลูกค้าเห็นสถานะคุณแบบเรียลไทม์
- **คะแนนเฉลี่ย:** ใต้ตัวนับเวลาจะแสดงคะแนนเฉลี่ยจากงานที่ลูกค้าประเมิน

---

### 1.4 แท็บงาน (Tasks) 📋

#### ดูรายการงาน
- เลือกแท็บ **งาน** จะเห็นรายการงานที่ลูกค้ามอบหมายให้คุณ
- แต่ละงานแสดง: **เนื้อหางาน (ไทย+เกาหลี)**, **สถานะ**, **วันครบกำหนด**

#### สีวันครบกำหนด
| สี | ความหมาย |
|----|----------|
| 🔴 แดง | เลยกำหนดแล้ว หรือใกล้หมดเวลา |
| 🟠 ส้ม | กำลังจะถึงกำหนด |
| ⚪ เทา | ยังมีเวลาเหลือ |

#### ทำเครื่องหมาย "เสร็จแล้ว"
1. เลือกงานที่ทำเสร็จ
2. คลิกปุ่ม **เสร็จแล้ว (완료)** — สถานะจะอัปเดตไปที่ฝั่งลูกค้าทันที

#### เสนองาน (Propose Task)
คุณสามารถ **เสนองานเอง** ให้ลูกค้าพิจารณาได้:
1. คลิกปุ่ม **"+ เสนองาน"** ด้านบนของรายการ
2. พิมพ์เนื้อหางาน **เป็นภาษาไทย** — ระบบจะแปลเป็นเกาหลีอัตโนมัติ
3. กดส่ง — งานจะปรากฏในระบบพร้อมป้าย **"เสนอเอง (자체 제안)"**

> ถ้ามีงานที่ลูกค้าสั่งยังไม่เสร็จ ระบบจะแจ้งเตือน (แต่ยังส่งงานเสนอได้)

---

### 1.5 แท็บแชท (Chat) 💬

แท็บแชทแบ่งเป็น 2 ส่วน:

#### ห้องแชทรวม (전체 톡방)
- ห้องแชทกลุ่มสำหรับสื่อสารกับลูกค้าและเพื่อนร่วมงานทุกคน
- พิมพ์ข้อความภาษาไทย → ระบบแปลเป็นเกาหลีอัตโนมัติ
- ข้อความจากลูกค้า (เกาหลี) จะแปลเป็นไทยให้
- แสดง **ชื่อผู้ส่ง** ทุกข้อความ

#### แชทตามงาน (업무별 채팅)
- เลือกงานจากรายการ → เปิดแชทเฉพาะงานนั้น
- ใช้สื่อสารรายละเอียดเกี่ยวกับงานนั้นๆ โดยเฉพาะ
- การแปลภาษาทำงานเหมือนห้องแชทรวม

#### การแนบไฟล์
- คลิกไอคอน 📎 ข้างช่องพิมพ์ข้อความ
- เลือกไฟล์ที่ต้องการแนบ (รูปภาพ, เอกสาร ฯลฯ)
- ไฟล์จะถูกส่งพร้อมข้อความ

#### การ @mention
- พิมพ์ **@** ตามด้วยชื่อเพื่อนร่วมงานเพื่อแจ้งเตือนเฉพาะบุคคล

---

### 1.6 แท็บแปล (Translation) 🌐

- ใช้สำหรับ **แปลข้อความไทย → เกาหลี** ทันที
- พิมพ์ข้อความภาษาไทยในช่อง แล้วกดแปล
- ใช้ช่วยร่างข้อความก่อนนำไปใช้ในที่อื่น

---

### 1.7 การแจ้งเตือน (Notifications) 🔔

- เมื่อมี **งานใหม่** หรือ **ข้อความใหม่** ที่มอบหมายให้คุณ จะมีการแจ้งเตือนบนหน้าจอ (Push Notification)
- แม้จะไม่ได้เปิดแอปอยู่ก็จะได้รับการแจ้งเตือน

---

## ส่วนที่ 2: Chrome Extension (ส่วนขยายเบราว์เซอร์)

### 2.1 การติดตั้ง

1. เปิด Chrome แล้วไปที่ `chrome://extensions`
2. เปิด **โหมดนักพัฒนา (Developer mode)** ทางมุมขวาบน
3. คลิก **โหลดส่วนขยายที่คลายการบีบอัดแล้ว (Load unpacked)**
4. เลือกโฟลเดอร์ `dist` ของ SyncBridge Extension

> หรือติดตั้งจาก Chrome Web Store (ถ้ามี) โดยค้นหา "SyncBridge"

### 2.2 การเข้าสู่ระบบ

1. คลิกไอคอน SyncBridge บน Chrome toolbar
2. ใส่ **อีเมล** และ **รหัสผ่าน** ที่ได้รับจากผู้ดูแล
3. คลิกเข้าสู่ระบบ

---

### 2.3 การลงเวลาทำงาน

เหมือนกับ Desktop App — มีปุ่ม 3 แบบ:
- **เข้างาน** 🟢 / **ไม่อยู่** 🟡 / **เลิกงาน** ⚪
- มีตัวนับเวลาแสดงระยะเวลาที่อยู่สถานะนั้น
- **Activity Ping:** ถ้าเข้างานอยู่แต่ไม่ขยับเมาส์/คีย์บอร์ดเกิน **10 นาที** ระบบจะเปลี่ยนเป็น "ไม่อยู่" อัตโนมัติ

---

### 2.4 แท็บงาน (Tasks) 📋

เหมือนกับ Desktop App:
- ดูรายการงาน + สีวันครบกำหนด
- ทำเครื่องหมาย "เสร็จแล้ว"
- **เสนองาน (Propose Task)** — เสนองานเองให้ลูกค้าพิจารณา

---

### 2.5 แท็บแชท (Chat) 💬

- แชทตามงาน — เลือกงานจากรายการ แล้วพิมพ์ข้อความ
- ภาษาไทยที่คุณพิมพ์จะถูกแปลเป็นเกาหลีอัตโนมัติ
- ข้อความจากลูกค้าจะแสดงเป็นเกาหลี (พร้อมคำแปลไทย)

---

### 2.6 แท็บแปล (Translation) 🌐

- แปลข้อความ **ไทย → เกาหลี** ทันที
- ใช้ช่วยร่างข้อความก่อนนำไปใช้

---

### 2.7 แท็บเทมเพลต (Templates) 📝

- โหลด **ข้อความตอบกลับสำเร็จรูป** ที่ลูกค้า/ผู้ดูแลตั้งไว้
- คลิกเทมเพลตที่ต้องการ → ข้อความจะถูก **คัดลอกไปที่คลิปบอร์ด**
- นำไปวางในแชทหรือที่อื่นได้ทันที

---

### 2.8 AI ดราก์ — วิเคราะห์ข้อความด้วย AI 🤖

ฟีเจอร์พิเศษของ Extension สำหรับช่วยตอบลูกค้า:

1. **ลากเลือกข้อความ** บนเว็บไซต์ที่คุณทำงานอยู่ (เช่น แชทโรงพยาบาล)
2. จะมีปุ่ม **"AI"** ลอยขึ้นมาข้างๆ ข้อความที่เลือก
3. คลิกปุ่ม **AI** → ระบบจะ:
   - 📝 แปลข้อความเป็น **ภาษาเกาหลี**
   - 🔍 วิเคราะห์ว่าลูกค้า/คนไข้ **ต้องการอะไร**
   - 💡 แสดง **คำตอบแนะนำ 3 แบบ** (ภาษาไทย)
4. คลิกคำตอบที่ต้องการ → ข้อความจะถูก **คัดลอกไปที่คลิปบอร์ด** พร้อมแจ้งเตือน

> ใช้ฟีเจอร์นี้ช่วยตอบคำถามลูกค้าได้เร็วขึ้นและตรงประเด็น

---

### 2.9 การแจ้งเตือนและไอคอนแบดจ์ 🔔

- เมื่อมี **งานใหม่** ไอคอน SyncBridge บน Chrome จะแสดง **ตัวเลขแจ้งเตือน (Badge)**
- เปิด Extension เพื่อดูรายการงานและแชทล่าสุด

---

## ส่วนที่ 3: เปรียบเทียบ Desktop App vs Extension

| ฟีเจอร์ | Desktop App | Chrome Extension |
|---------|:-----------:|:----------------:|
| เข้างาน/ไม่อยู่/เลิกงาน | ✅ | ✅ |
| ดูรายการงาน | ✅ | ✅ |
| ทำเครื่องหมาย "เสร็จแล้ว" | ✅ | ✅ |
| เสนองาน | ✅ | ✅ |
| แชทตามงาน | ✅ | ✅ |
| ห้องแชทรวม | ✅ | ❌ |
| แปลภาษา | ✅ | ✅ |
| แนบไฟล์ | ✅ | ❌ |
| @mention | ✅ | ❌ |
| เทมเพลตตอบกลับ | ❌ | ✅ |
| AI ดราก์ (วิเคราะห์) | ❌ | ✅ |
| Activity Ping อัตโนมัติ | ❌ | ✅ (10 นาที) |
| แจ้งเตือน Push | ✅ | ✅ (Badge) |
| อัปเดตอัตโนมัติ | ✅ | ❌ (ต้องอัปเดตเอง) |

> **แนะนำ:** ใช้ **Desktop App** เป็นหลักสำหรับการรับงานและแชท ใช้ **Extension** เป็นตัวช่วยเสริมเมื่อทำงานบนเว็บ (เช่น AI ดราก์ วิเคราะห์ข้อความ)

---

## ส่วนที่ 4: สรุปขั้นตอนการทำงานประจำวัน

### เริ่มงาน
1. เปิด **Desktop App** (หรือ Extension)
2. เข้าสู่ระบบ (ถ้ายังไม่ได้ล็อกอิน)
3. กดปุ่ม **เข้างาน** 🟢

### ระหว่างวัน
4. ดูแท็บ **งาน** — ตรวจสอบงานที่ได้รับมอบหมาย
5. ทำงานตามที่ได้รับ → กด **เสร็จแล้ว** เมื่อทำครบ
6. ใช้ **แชท** สื่อสารกับลูกค้าเกี่ยวกับงาน (ข้อความจะแปลอัตโนมัติ)
7. ใช้ **ห้องแชทรวม** (Desktop App) สำหรับการสื่อสารทั่วไป
8. ใช้ **แปล** เมื่อต้องการแปลข้อความเพิ่มเติม
9. ใช้ **เทมเพลต** (Extension) สำหรับข้อความตอบกลับที่ใช้บ่อย
10. ใช้ **AI ดราก์** (Extension) เมื่อต้องการวิเคราะห์ข้อความลูกค้าบนเว็บ
11. **เสนองาน** เมื่อมีไอเดียงานที่อยากทำ

### เลิกงาน
12. กดปุ่ม **เลิกงาน** ⚪ — ระบบจะบันทึกเวลาอัตโนมัติ

## คำถามที่พบบ่อย

### ถาม: ลืมรหัสผ่าน ทำอย่างไร?
ตอบ: ติดต่อผู้ดูแลระบบ (BBG Admin) เพื่อรีเซ็ตรหัสผ่าน

### ถาม: สถานะเปลี่ยนเป็น "ไม่อยู่" เอง ทำไม?
ตอบ: ระบบ Activity Ping ของ Extension จะเปลี่ยนสถานะอัตโนมัติถ้าไม่มีการใช้งานเกิน 10 นาที กดปุ่ม "เข้างาน" อีกครั้งเพื่อกลับมาทำงาน

### ถาม: ข้อความแปลไม่ถูกต้อง ทำอย่างไร?
ตอบ: การแปลใช้ AI (Google Gemini) ซึ่งอาจไม่สมบูรณ์ 100% หากข้อความสำคัญ ให้ใช้แท็บ "แปล" แปลแยกก่อน แล้วแก้ไขก่อนส่ง

### ถาม: แอป Desktop ขึ้นข้อความว่ามีเวอร์ชันใหม่ ต้องทำอย่างไร?
ตอบ: คลิก "อัปเดต" หรือ "ติดตั้งและรีสตาร์ท" ระบบจะดาวน์โหลดและติดตั้งเวอร์ชันใหม่ให้อัตโนมัติ

### ถาม: ห้องแชทรวมไม่แสดง ทำอย่างไร?
ตอบ: ตรวจสอบว่าโปรไฟล์ของคุณได้ตั้งค่า client_id (รหัสโรงพยาบาล) แล้ว หากไม่แน่ใจ กรุณาติดต่อผู้ดูแลระบบ
