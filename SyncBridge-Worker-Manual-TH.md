# คู่มือการใช้งาน SyncBridge สำหรับพนักงาน (Worker)

**SyncBridge** — ระบบซิงค์งานระหว่างบริษัทลูกค้า (เกาหลี) กับพนักงานทำงานทางไกล (ไทย)  
คู่มือฉบับนี้สำหรับ **พนักงาน (Worker)** ที่ใช้ Chrome Extension ในการรับ-ส่งงานและแชทกับลูกค้า

---

## 1. การติดตั้งและเข้าสู่ระบบ

### 1.1 ติดตั้ง Chrome Extension

1. เปิด Chrome แล้วไปที่ `chrome://extensions`
2. เปิด **โหมดนักพัฒนา (Developer mode)** ทางมุมขวาบน
3. คลิก **โหลดส่วนขยายที่คลายการบีบอัดแล้ว (Load unpacked)**
4. เลือกโฟลเดอร์ `dist` ของ SyncBridge Extension

### 1.2 เข้าสู่ระบบ

1. คลิกไอคอน SyncBridge บน Chrome
2. ใส่ **อีเมล** และ **รหัสผ่าน** ที่ได้รับจากผู้ดูแล
3. คลิกเข้าสู่ระบบ — หลังล็อกอินระบบจะสร้างโปรไฟล์ Worker ให้อัตโนมัติ

---

## 2. การลงเวลาทำงาน (ออก근 / ไม่อยู่ / เลิกงาน)

ที่ส่วนหัวของ Extension จะมีปุ่มสถานะ 3 แบบ:

| ปุ่ม | ความหมาย | การทำงาน |
|------|----------|----------|
| **ออก근 (Online)** | กำลังทำงาน | กดเพื่อบันทึกว่าออก근แล้ว — เวลาจะถูกบันทึกในระบบทันที |
| **ไม่อยู่ (Away)** | ลุกจากที่นั่งชั่วคราว | กดเมื่อต้องลุกไปทำอย่างอื่นสั้นๆ |
| **เลิกงาน (Offline)** | เลิกทำงานแล้ว | กดเมื่อจบกะ — ระบบจะบันทึกเวลเลิกงาน |

- **ตัวจับเวลา:** ใต้ปุ่มจะมีตัวนับเวลาแสดงระยะเวลาที่อยู่สถานะนั้น (อัปเดตทุกวินาที)
- **Activity Ping:** ถ้าคุณออก근อยู่แต่ไม่ขยับเมาส์/คีย์บอร์ดเกิน **10 นาที** ระบบจะเปลี่ยนสถานะเป็น "ไม่อยู่" อัตโนมัติ

---

## 3. แท็บ **งาน (Tasks)**

### 3.1 ดูรายการงานที่ได้รับมอบหมาย

- เลือกแท็บ **งาน (งาน)** จะเห็นรายการงานที่ลูกค้ามอบหมายให้คุณ
- แต่ละงานแสดง: **ชื่อ/เนื้อหา**, **สถานะ**, **วันครบกำหนด**
- **สีวันครบกำหนด:**
  - **แดง** — ใกล้หมดเวลาหรือเลยกำหนด
  - **ส้ม** — กำลังจะถึงกำหนด
  - **เทา** — ยังมีเวลาพอ

### 3.2 ทำเครื่องหมายงานเป็น "เสร็จแล้ว"

1. เลือกงานที่ทำเสร็จแล้ว
2. คลิกปุ่ม **เสร็จแล้ว / Complete** — สถานะจะอัปเดตไปที่ฝั่งลูกค้าแบบเรียลไทม์

### 3.3 เสนองาน (เสนองานด้วยตัวเอง — Propose Task)

คุณสามารถ **เสนองานที่อยากทำ** ให้ลูกค้าพิจารณาได้:

1. ที่แท็บ **งาน** คลิกปุ่ม **"+ เสนองาน (제안)"** ด้านบน
2. กรอก **เนื้อหางานเป็นภาษาไทย** — ระบบจะแปลเป็นเกาหลีให้อัตโนมัติ
3. กดส่ง — งานจะถูกบันทึกในระบบพร้อมป้าย **"เสนอเอง (자체 제안)"**
4. ถ้ามีงานที่ลูกค้าสั่งยังไม่เสร็จ ระบบจะแจ้งเตือน (แต่ยังส่งงานเสนอได้)

---

## 4. แท็บ **แชท (Chat)**

- เลือกแท็บ **แชท** เพื่อดูการสนทนาตามแต่ละงาน
- เลือกงานที่ต้องการจากรายการ แล้วพิมพ์ข้อความ
- **ภาษาไทยที่คุณพิมพ์จะถูกแปลเป็นเกาหลีอัตโนมัติ** แล้วส่งไปยังลูกค้า
- ข้อความจากลูกค้าจะแสดงเป็นภาษาเกาหลี (และอาจมีคำแปลตามที่ระบบรองรับ)

---

## 5. แท็บ **แปล (Translation)**

- ใช้สำหรับ **แปลข้อความไทย → เกาหลี** ทันที
- พิมพ์ข้อความภาษาไทยในช่อง แล้วกดแปล — จะได้ประโยคภาษาเกาหลี
- ใช้ช่วยร่างข้อความก่อนนำไปใช้ในแชทหรือรายงานงาน

---

## 6. แท็บ **เทมเพลต (Templates / Quick Reply)**

- แท็บนี้โหลด **ข้อความตอบกลับสำเร็จรูป** จากระบบ (ที่ลูกค้า/ผู้ดูแลตั้งไว้)
- คลิกที่เทมเพลตที่ต้องการ → ข้อความจะถูก ** copy ไปที่คลิปบอร์ด**
- นำไปวางในแชทหรือที่อื่นได้ทันที

---

## 7. AI ดราก์ (วิเคราะห์ข้อความด้วย AI)

เมื่อคุณ **ทำงานบนเว็บไซต์** (เช่น ตอบลูกค้าในแชทของโรงพยาบาล):

1. **ลากเลือกข้อความ** ที่ต้องการวิเคราะห์ (อย่างน้อย 5 ตัวอักษร)
2. จะมีปุ่ม **"AI"** ลอยขึ้นมา
3. คลิกปุ่ม **AI** — ระบบจะ:
   - แปลข้อความเป็นภาษาเกาหลี
   - วิเคราะห์ว่าลูกค้าต้องการอะไร
   - แสดง **คำตอบแนะนำ 3 แบบ (ภาษาไทย)**
4. คลิกคำตอบที่ต้องการ → ข้อความจะถูก copy ไปที่คลิปบอร์ด พร้อมแจ้งเตือน

ใช้ฟีเจอร์นี้ช่วยตอบคำถามลูกค้าได้เร็วและสอดคล้องกับบริบท

---

## 8. การแจ้งเตือนและไอคอน

- เมื่อมี **งานใหม่** ที่มอบหมายให้คุณ ไอคอน SyncBridge บน Chrome จะแสดง **ตัวเลขแจ้งเตือน (Badge)**
- เปิด Extension เพื่อดูรายการงานและแชทล่าสุด

---

## สรุปการใช้งานประจำวัน

1. **เปิด Chrome** → คลิกไอคอน SyncBridge → **ออก근** เมื่อเริ่มทำงาน
2. ดูแท็บ **งาน** — ทำตามงานที่มอบหมาย และกด **เสร็จแล้ว** เมื่อทำครบ
3. ใช้แท็บ **แชท** สื่อสารกับลูกค้าตามงาน (ข้อความไทยจะถูกแปลเป็นเกาหลีอัตโนมัติ)
4. ใช้ **เทมเพลต** และ **AI ดราก์** ช่วยตอบลูกค้าให้เร็ว
5. ใช้ **แปล** เมื่อต้องการแปลประโยคไทย–เกาหลี
6. **เสนองาน** ได้เมื่อมีไอเดียงานที่อยากเสนอให้ลูกค้า
7. เมื่อจบกะ กด **เลิกงาน** — ระบบจะบันทึกเวลาอัตโนมัติ
