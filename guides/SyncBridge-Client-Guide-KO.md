# SyncBridge 사용 가이드 — 고객사(병원) 관리자용

**SyncBridge**는 한국 고객사(병원)와 태국 원격 근무자 간의 실시간 업무 동기화 플랫폼입니다.
이 가이드는 **병원 관리자(client)** 역할로 웹 대시보드를 사용하는 방법을 안내합니다.

---

## 1. 로그인

1. 웹 브라우저에서 SyncBridge 대시보드 URL에 접속합니다
2. BBG 관리자로부터 전달받은 **이메일**과 **비밀번호**를 입력합니다
3. **로그인** 버튼을 클릭합니다

> 계정은 BBG 관리자가 생성해 드립니다. 직접 가입은 불가합니다.

---

## 2. 대시보드 구성

로그인 후 대시보드에 아래 섹션들이 순서대로 표시됩니다.

| 순서 | 섹션 | 색상 | 설명 |
|------|------|------|------|
| 1 | 직원 상태 | 🔵 파란색 | 소속 직원의 실시간 근무 상태 |
| 2 | 전체 톡방 | 🟣 인디고 | 소속 직원 전체와의 그룹 채팅 |
| 3 | 업무 할당 | 🟢 초록색 | 직원에게 새 업무 지시 |
| 4 | 업무 목록 | 🟡 노란색 | 모든 업무 현황 + 채팅 + 평가 |
| 5 | 업무 캘린더 | 🟣 보라색 | 월별 업무 일정 현황 |
| 6 | 근무 리포트 | 🔵 청록색 | 오늘의 직원 근태 요약 |

---

## 3. 직원 상태 모니터링

대시보드 최상단에서 소속 직원들의 **실시간 근무 상태**를 확인할 수 있습니다.

| 상태 | 의미 |
|------|------|
| **온라인** (초록) | 현재 근무 중 |
| **자리비움** (노랑) | 잠시 자리를 비운 상태 |
| **오프라인** (회색) | 퇴근 또는 미접속 |

- 각 직원 카드에 **평균 품질 평점**이 표시됩니다 (1~5점)
- 상태는 실시간으로 자동 업데이트됩니다

---

## 4. 전체 톡방

소속 직원 전체와 소통할 수 있는 **그룹 채팅방**입니다.

### 사용 방법
1. **전체 톡방** 섹션에서 메시지를 입력합니다
2. **한국어로 입력**하면 태국어로 자동 번역되어 직원에게 전달됩니다
3. 직원이 태국어로 보낸 메시지는 한국어 번역과 함께 표시됩니다

### 참고
- 접기/펼치기 버튼으로 톡방을 접을 수 있습니다
- 각 메시지에 **발신자 이름**이 표시됩니다
- 메시지는 실시간으로 수신됩니다

### 파일 첨부
- 톡방 입력창 옆의 📎 버튼을 클릭하여 파일을 첨부할 수 있습니다
- **지원 파일:** 이미지(JPG, PNG 등), PDF, 문서(DOC, XLS, PPT), TXT, ZIP
- **크기 제한:** 10MB 이하
- 이미지는 채팅 내에서 **미리보기**로 표시되며, 클릭하면 크게 볼 수 있습니다
- 문서 파일은 파일명과 다운로드 링크로 표시됩니다

### @멘션 (태그)
- 메시지 입력 중 **@**를 입력하면 팀원 목록이 나타납니다
- 원하는 팀원을 선택하면 @이름이 자동 입력됩니다
- 멘션된 메시지는 **하이라이트**로 표시되어 쉽게 구분됩니다
- Desktop App 사용자에게는 **푸시 알림**이 전송됩니다

---

## 5. 업무 할당

직원에게 새로운 업무를 지시합니다.

### 기본 할당
1. **담당자**를 선택합니다 (드롭다운에서 소속 직원 목록)
2. **업무 내용**을 한국어로 입력합니다
3. 필요시 **마감일**을 설정합니다 (날짜 선택)
4. **할당** 버튼을 클릭합니다

> 한국어로 입력한 업무 내용은 **태국어로 자동 번역**되어 직원에게 전달됩니다.

### 프리셋 사용
자주 반복하는 업무는 **프리셋**으로 빠르게 할당할 수 있습니다.

1. 업무 할당 폼 상단의 **프리셋 드롭다운**을 클릭합니다
2. 원하는 프리셋을 선택하면 업무 내용이 자동으로 채워집니다
3. 필요시 내용을 수정한 후 할당합니다

> 프리셋은 BBG 관리자가 등록·관리합니다. 병원 전용 프리셋과 전체 공용 프리셋이 있습니다.

---

## 6. 업무 목록

등록된 모든 업무의 현황을 확인하고 관리합니다.

### 업무 상태
| 상태 | 의미 |
|------|------|
| **대기** | 직원에게 전달됨, 아직 시작 안 함 |
| **진행 중** | 직원이 작업 중 |
| **완료** | 직원이 완료 처리함 |

### 마감일 색상 코딩
| 색상 | 의미 |
|------|------|
| 🔴 빨간색 | 마감일 초과 또는 임박 |
| 🟠 주황색 | 마감일이 가까움 |
| ⚪ 회색 | 여유 있음 |

### 업무별 채팅
각 업무 항목을 클릭하면 해당 업무에 대한 **1:1 채팅**을 할 수 있습니다.
- 한국어로 입력 → 태국어로 자동 번역되어 직원에게 전달
- 직원의 태국어 메시지는 한국어 번역과 함께 표시
- 📎 버튼으로 파일 첨부 가능 (이미지, 문서 등)
- @를 입력하여 팀원 멘션 가능

### 품질 평가 (별점)
직원이 업무를 완료하면 **1~5점 별점**으로 품질을 평가할 수 있습니다.
1. 완료된 업무 항목에 표시되는 ⭐ 아이콘을 클릭합니다
2. 1~5점 중 점수를 선택합니다
3. 평가는 즉시 저장되며, 직원 카드의 평균 평점에 반영됩니다

---

## 7. 업무 캘린더

월별 달력 형태로 업무 일정을 한눈에 확인합니다.

- 각 날짜에 **해당일 업무 수**가 표시됩니다
- 월 이동 버튼(◀ ▶)으로 다른 달을 확인할 수 있습니다
- 업무가 많은 날은 숫자가 크게 표시되어 일정 밀도를 파악할 수 있습니다

---

## 8. 자동답변 관리 (Quick Reply)

직원들이 자주 사용하는 **응답 템플릿**을 등록·관리합니다.

### 등록 방법
1. 자동답변 관리 섹션에서 **추가** 버튼을 클릭합니다
2. **제목**과 **본문**을 한국어로 입력합니다
3. 저장하면 태국어로 자동 번역됩니다

### 활용
- 등록한 자동답변은 직원의 Extension **템플릿 탭**에 표시됩니다
- 직원이 템플릿을 클릭하면 바로 복사하여 사용할 수 있습니다
- 병원별 자동답변과 전체 공용 자동답변을 구분하여 관리할 수 있습니다

---

## 9. 근무 리포트

오늘 하루의 직원 근태 현황을 요약합니다.

| 항목 | 설명 |
|------|------|
| 출근 시간 | 직원이 "출근" 버튼을 누른 시간 |
| 퇴근 시간 | 직원이 "퇴근" 버튼을 누른 시간 |
| 근무 시간 | 총 근무 시간 (자리비움 제외) |
| 출근율 | 프로그레스 바로 시각적 표시 |

- 각 직원별로 색상 코딩된 근태 정보를 한눈에 확인할 수 있습니다

---

## 10. 번역 안내

SyncBridge의 모든 채팅은 **자동 번역**됩니다.

### 동작 방식
1. 한국어로 메시지를 입력하면 **즉시 전송**됩니다
2. 백그라운드에서 태국어로 번역이 진행됩니다
3. 번역이 완료되면 직원 화면에 태국어가 자동 표시됩니다

> 번역은 Google Gemini AI를 사용하며, 원본 메시지는 항상 보존됩니다.

---

## 11. 직원의 업무 제안

직원이 직접 업무를 제안할 수도 있습니다.

- 직원이 제안한 업무는 업무 목록에 **"자체 제안"** 뱃지로 표시됩니다
- 일반 업무와 동일하게 확인·채팅·평가가 가능합니다

---

## 자주 묻는 질문

### Q. 직원 상태가 업데이트되지 않아요
직원이 Extension 또는 Desktop App에 로그인되어 있는지 확인해 주세요. 직원이 10분 이상 무활동 시 자동으로 "자리비움"으로 변경됩니다.

### Q. 전체 톡방이 보이지 않아요
직원의 프로필에 **소속 병원(client_id)**이 설정되어 있어야 합니다. BBG 관리자에게 확인을 요청해 주세요.

### Q. 번역이 안 돼요
번역은 메시지 전송 후 몇 초 내에 완료됩니다. 잠시 기다려 주세요. 계속 번역되지 않으면 BBG 관리자에게 문의해 주세요.

### Q. 새 직원 계정을 만들고 싶어요
직원 계정은 BBG 관리자만 생성할 수 있습니다. 담당 BBG 관리자에게 요청해 주세요.

### Q. 직원에게 메시지를 보냈는데 확인을 안 해요
Desktop App을 사용하는 직원에게는 새 메시지 수신 시 **푸시 알림**이 전송됩니다. @멘션을 사용하면 특정 직원에게 직접 알림이 갑니다.
