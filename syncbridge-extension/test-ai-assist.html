🤖 SyncBridge AI Assist 테스트

아래 텍스트를 드래그(블록 지정)하면 ✨ AI 버튼이 나타납니다.

테스트 케이스 1: 태국어 환자 메시지

สวัสดีครับ ผมต้องการสอบถามเกี่ยวกับการนัดหมาย ผมอยากทราบว่ามีเวลาว่างในสัปดาห์นี้ไหมครับ

예상: ✨ AI 버튼 → 클릭 → Mock 응답 팝업

테스트 케이스 2: 짧은 텍스트 (5자 미만 - 버튼 안 나타남)

สวัสดี

예상: 버튼 안 나타남 (5자 이하 무시)

테스트 케이스 3: 공백만 있는 텍스트

예상: 버튼 안 나타남 (공백 제외 후 길이 체크)

테스트 케이스 4: 긴 환자 문의

สวัสดีครับ ผมมีคำถามเกี่ยวกับการผ่าตัดที่โรงพยาบาล ผมต้องการทราบข้อมูลเกี่ยวกับค่าใช้จ่ายและขั้นตอนการเตรียมตัวก่อนผ่าตัดครับ ขอบคุณครับ

예상: ✨ AI 버튼 → 클릭 → 분석 결과

검증 체크리스트

  • ✅ 텍스트 드래그 시 ✨ AI 버튼이 커서 근처에 나타나는가?
  • ✅ 버튼 클릭 시 로딩 애니메이션이 나타나는가?
  • ✅ Mock 응답 팝업이 나타나는가? (API 없어도 동작)
  • ✅ 추천 답변 클릭 시 클립보드에 복사되는가?
  • ✅ 팝업 외부 클릭 시 닫히는가?
  • ✅ 팝업 내부 클릭 시 닫히지 않는가?
  • ✅ Shadow DOM이 Host 페이지 CSS와 충돌하지 않는가?
